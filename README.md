# uno-api
Simple UNO game written in FastAPI

# Api Reference
See `http://localhost:3000/redoc`

# Run Locally
Clone the project

```bash
    git clone https://github.com/0x16c3/uno-api
```

Go to the project directory

```bash
    cd uno-api
```

Install dependencies

```bash
    pip install -r requirements.txt
```

Start the server

```bash
    py server.py
```
